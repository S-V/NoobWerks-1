1) SYSTEM REQUIREMENTS
_______________________________________________________________________

System Requirements

    Minimum:
        OS: Windows 7 or later
        Processor: 2-Core 2 GHz
        Memory: 512 MB RAM
        Graphics: DX11 compatible GPU with 1 GB VRAM
        DirectX: Version 11
        Storage: 100 MB available space
		Keyboard
		Mouse
		
    Recommended:
        OS: Windows 7 or later
        Processor: Intel Core i5, i7 3.2 GHz or AMD FX, Athlon 3.6 GHz or equivalent
        Memory: 1 GB RAM
        Graphics: DX11 compatible GPU with 1 GB VRAM
        DirectX: Version 11
        Storage: 512 MB available space
		Keyboard
		Mouse

8) ACKNOWLEDGEMENTS
_______________________________________________________________________

HALF-ASSED uses assets from the following games:
- Doom3 (models, animations, textures, sounds)
- Project I.G.I. (weapon pick up sounds)
- Nexuiz (gibs models)
- Rebel Moon/Rebel Moon Rising (background music).

HALF-ASSED is linked with the following libraries (in no particular order):

- Bullet Physics: https://github.com/bulletphysics/bullet3
- ozz-animation: https://github.com/guillaumeblanc/ozz-animation
- Dear ImGui: https://github.com/ocornut/imgui
- ImGuizmo: https://github.com/CedricGuillemet/ImGuizmo
- assimp: https://github.com/assimp/assimp
- SDL 2.0: https://www.libsdl.org/
- gainput: https://github.com/jkuhlmann/gainput
- LZ4: https://github.com/lz4/lz4
- LMDB: https://github.com/LMDB/lmdb
- SQLite: https://www.sqlite.org/index.html
- DebugHeap: https://github.com/deplinenoise/ig-debugheap
- nativefiledialog: https://github.com/mlabbe/nativefiledialog
